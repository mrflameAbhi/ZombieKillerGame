# ZombieKillerGame
Free To Use No need to Give the Credit
"Zombie Killer" is first-person shooter game developed using Unreal Engine 5, Set on a bridge on which the player character has to kill the zombies to survive. The Zombies come to attack player on the bridge and the spawn on the bridge on the wave system . Total waves are three and In the first wave 4 zombies are spawned on the front and behind of the  player.Player have to kill all 4 zombies to go to the next wave.
Overall The Game is made for entertainment purpose of the user and It is user friendly and easy levels to clear.
